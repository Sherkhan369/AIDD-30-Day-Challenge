# PART A — Research Questions

1. **What new improvements were introduced in Gemini 3.0?**

Gemini 3 mein reasoning (sochne ki salahiyat) ko bohot improve kiya gaya hai. Iske sath ek naya feature "Deep Think" introduce kiya gaya hai, jo mushkil scientific aur math ke problems solve karne ke liye model ko gehraai se sochne (step-by-step reasoning) ki taqat deta hai.

2. **How does Gemini 3.0 improve coding & automation workflows?**

Developers ke liye ek naya platform "Google Antigravity" launch kiya gaya hai. Ye developers ko aise AI agents banane mein madad karta hai jo coding aur doosre complex tasks khud perform kar sakein.

3. **How does Gemini 3.0 improve multimodal understanding?**

Ye model text, images, audio, video aur code ko ek hi waqt mein samajh sakta hai. Purane models ke muqable iski "native multimodality" bohot teez aur accurate hai.


4. **Name any two developer tools introduced with Gemini 3.0.**
- Google Antigravity
- Vertex AI Agent Builder (with Gemini 3.0)
